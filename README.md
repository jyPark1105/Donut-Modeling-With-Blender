# Donut-Modeling-With-Blender
Creating a single donut in Blender and duplicating it to make multiple copies

# **Learn the parts of 3D modeling by creating a donut using the features of the Blender program**
This repository is based on the 3D Modeling tutorial videos by **Blender Guru** and was created to study computer graphics. Each project showcases 3D work in Blender, including creating a single donut and modeling multiple replicas of it. And here is the site.("https://www.youtube.com/watch?v=B0J27sf9N1Y&list=PLjEaoINr3zgEPv5y--4MKpciLaoQYZB1Z")
